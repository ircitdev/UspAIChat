# UspAIChat — Ценовая политика и финансовая модель

> Последнее обновление: 2026-03-23

## 1. Базовые принципы

- **1 кредит = 1 рубль** (при пополнении через ЮKassa)
- Списание только за **выходные токены** (текст, генерируемый моделью)
- Стоимость **входных токенов** (запрос пользователя + история) заложена в наценку
- Администраторы не расходуют кредиты
- Все цены можно переопределить через админку (settings DB)

## 2. Курс расчёта

- **Курс USD/RUB**: ~90 ₽/$
- **Средний ratio input:output**: 2.5:1 (на основе типичных диалогов)
- **Формула**: `Effective cost = API_output_₽ + API_input_₽ × 2.5`
- **Целевая наценка**: x2.5 (~150% прибыли)

## 5. Примеры расхода для пользователей

### Что можно получить за 100 кредитов (100 ₽):

| Сценарий | Модель | Кол-во сообщений |
|----------|--------|-----------------|
| Простые вопросы | Gemini 2.0 Flash (0.15 кр) | ~660 ответов |
| Быстрые ответы | GPT-4o mini (0.3 кр) | ~330 ответов |
| Обычные диалоги | Claude Sonnet (5.0 кр) | ~20 ответов |
| Код/анализ | GPT-4o (4.0 кр) | ~25 ответов |
| Сложные задачи | Claude Opus (25.0 кр) | ~4 ответа |
| Дешёвый AI | DeepSeek V3 (0.4 кр) | ~250 ответов |

*Расчёт: средний ответ ~1K токенов*

### Типичный сценарий: 5 сообщений в день, Claude Sonnet

- Средний ответ: ~500 токенов
- Стоимость: 5 × 0.5K × 5.0 = **12.5 кр/день**
- В месяц: 12.5 × 30 = **375 кр/месяц**
- Пакет «Стандарт» (500₽ → 525 кр) хватит на **~42 дня**

### Smart Router экономия

При использовании Auto-режима, простые запросы идут на дешёвые модели:
- Без роутера (все на Sonnet): 30 сообщ/день × 2.5 кр = **75 кр/день**
- С роутером (70% на Flash, 30% на Sonnet): 21×0.15 + 9×2.5 = **25.65 кр/день** (экономия ~66%)

## 7. Реферальная программа

| Параметр | Значение |
|----------|----------|
| Бонус рефереру | 50 кр + 10% от первого пополнения друга |
| Бонус другу | 25 кр при регистрации |
| Макс. стоимость привлечения | ~100 кр (при пополнении друга на 500₽) |
| Себестоимость бонусов | ~40 ₽ (40% от 100 кр) |
| LTV привлечённого | ~3 000 ₽ (6 месяцев × 500₽) |
| **ROI реферальной программы** | **~7 400%** |

## 8. Риски

| Риск | Влияние | Митигация |
|------|---------|-----------|
| Рост курса USD | Себестоимость растёт | Автоматический пересчёт цен через settings DB |
| Изменение цен API | Маржа снижается | Мониторинг + обновление DEFAULT_PRICES |
| Злоупотребление (длинные контексты) | Input >> Output, маржа ниже | Ratio 2.5:1 заложен с запасом |
| Конкуренция | Давление на цены | Дифференциация через Smart Router |
