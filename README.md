# UspAIChat

AI чат-приложение с поддержкой Anthropic Claude, OpenAI и Google Gemini.

## Запуск

1. Установите зависимости:
   ```bash
   npm run install:all
   ```

2. Запустите backend (терминал 1):
   ```bash
   npm run dev:backend
   ```

3. Запустите frontend (терминал 2):
   ```bash
   npm run dev:frontend
   ```

4. Откройте http://localhost:3000

5. Перейдите в Настройки (Ctrl+,) и добавьте API ключи.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Ctrl+N | Новый чат |
| Ctrl+K | Поиск |
| Ctrl+, | Настройки |
| Ctrl+B | Боковая панель |
| Enter | Отправить |
| Shift+Enter | Новая строка |
| Escape | Закрыть |
